npm - Node.js package manager

packages - self contained code libraries that we can install and use in our projects. Short for packaged library.

npm install blank

npm install with no args will install all dependencies listed on the JSON file

npm i

  • if you are not sure what the package name you want is, you can run npm info and the arg for seeing a list of npm repo in CLI

package.json

all Node.js project have a package.json file

ie.,

{
  "name": "project-name",
  "version": "1.0.0",
  "description": "Short project summary",
  "main": "index.js",
  "scripts": {
    "myscript": "ENV=development node index.js",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "MIT",
  "dependencies": {
    "express": "^4.13.4"
  }
}

  • scripts: - allows us to run commands using an alias

    npm run myscript- ie 'mocha' to run the tests using npm

  • dependencies: refers to the packages that need to be installed for the project to run, and their version numbers.

npm install adds a dependency to the package.json file

package-lock.json

  • lists all the details about our projects dependencies.

  • instead use npm install to make any changes
